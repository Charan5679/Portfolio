# Sri Charan Vagalagani — Portfolio

## ⚡ Start (Windows)

Just double-click **START.bat** — it cleans the cache, installs, and starts automatically.

**Or manually:**
```
rd /s /q .next
npm install --legacy-peer-deps
npm run dev
```

## ⚡ Start (Mac / Linux)
```
./start.sh
```

Open → http://localhost:3000

---

**Important:** Always delete the `.next` folder before running after a fresh download.
The `START.bat` / `start.sh` scripts do this automatically.

## Stack
Next.js 14 · React 18 · TypeScript · Three.js · Framer Motion · Tailwind · GSAP · Lenis
